# SECTION G — CODE & COMMAND READING (10 Marks)

### Question 34

What does this command do?

```bash
git init
```

**Answer:**

Creates a new Git repository in the current folder.

---

### Question 35

What does this command do?

```bash
git status
```

**Answer:**

Shows the current state of the repository and files.

---

### Question 36

What does this command do?

```bash
git add .
```

**Answer:**

Stages all modified and new files for commit.

---

### Question 37

What does this command do?

```bash
git commit -m "Added login feature"
```

**Answer:**

Creates a commit with the message "Added login feature".

# SECTION H — DEBUGGING QUESTIONS (5 Marks)

### Question 38

A student runs:

```bash
git commit -m "My update"
```

but receives:

```text
nothing to commit
```

What did they most likely forget?

**Answer:**

They forgot to make changes or run:

```bash
git add .
```

before committing.

---

### Question 39

A student accidentally uploads:

```text
venv/
```

to GitHub.

How can this be prevented?

**Answer:**

Add:

```text
venv/
```

to the `.gitignore` file.
