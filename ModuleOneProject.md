# Common Beginner Errors

## Error 1: Missing Closing Quote

Wrong:

```python
print("Hello World)
```

Output:

```text
SyntaxError
```

Correct:

```python
print("Hello World")
```

---

## Error 2: Missing Parenthesis

Wrong:

```python
print("Hello World"
```

Correct:

```python
print("Hello World")
```

---

## Error 3: Misspelling print

Wrong:

```python
prnt("Hello")
```

Correct:

```python
print("Hello")
```
